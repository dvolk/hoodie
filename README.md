# hoodie

A small, toy roguelike in Haskell

[Obligatory screenshot](http://i.imgur.com/b1dzGeZ.png)

## Instructions
### Downloading

    $ mkdir ~/src
    $ cd ~/src
    $ git clone https://github.com/dvolk/hoodie

### Building

    $ cd hoodie
    $ cabal configure
    $ cabal build

### Running

    $ ./dist/build/hoodie/hoodie
